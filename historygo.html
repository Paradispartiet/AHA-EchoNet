History Go i AHA

History Go er samlings- og læringsuniverset. AHA importerer bare utvalgte History Go-signaler som personlige AHA source events/insights. AHA skriver ikke tilbake til History Go-lagring som standard.

Status

Import boundary

  • AHA reads aha_import_payload_v1.
  • Manual import only.
  • Writes AHA source events/insights.
  • Does not write back to HG storage by default.
  • Database persist disabled by default.
  • EchoNet/sync disabled.

          Forklaring

          • History Go er samlings- og læringsuniverset.
          • AHA importerer bare utvalgte History Go-signaler som personlige AHA source events/insights.
          • AHA lager ikke ny History Go-motor.
          • AHA skriver ikke tilbake til History Go-lagring som standard.
          • Import skjer bare når brukeren trykker import.
          • Sync, backend og EchoNet er ikke aktivert.
